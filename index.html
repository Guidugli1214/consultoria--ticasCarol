⚙️ Sincronização com GitHub

As visitas serão salvas no arquivo visits.json do repositório do GitHub, permitindo acesso de qualquer dispositivo.

Crie um token em GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Permissão necessária: Contents → Read and write no repositório consultoria--ticasCarol.
